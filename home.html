Search all the Pokemon you need here, including which Pokemon has each move, ability, or stats.

If you wanted to search Pokemon by ability, move, or stats, this is the right place for you! This is a more streamlined way of searching Pokemon by those parameters, as sites like bulbapedia are somewhat less convenient for doing this because you need to search and change URLs.
There are 6 searchers implemented, which will be explained here:
Search all Pokemon: Search all the Pokemon by name.
Search by Abiliity: Search all the Pokemon by their ability.
Search by Move: Search all the Pokemon by the moves they learn.
All Abilities: list of all abilities.
All Moves: list of all moves.
